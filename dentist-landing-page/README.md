# Dr. Elena Bloom - Premier Dental Care Landing Page

A beautiful, modern landing page for a dental practice featuring a "refined wellness" aesthetic that combines medical professionalism with spa-like tranquility.

## Design Features

### Aesthetic Direction
- **Refined Wellness Theme**: Combines medical trust with spa-like tranquility
- **Organic Curves**: Flowing shapes and natural forms throughout
- **Sophisticated Color Palette**: Warm terra cotta, sage green, and cream
- **Premium Typography**: Crimson Text for elegance, Jost for modern clarity

### Key Components
- **Hero Section**: Gradient background with animated floating elements
- **Services Grid**: Card-based layout with hover animations
- **About Section**: Split layout with animated counters
- **Testimonials**: Rotating carousel with fade transitions
- **Contact Form**: Elegant styling with validation
- **Interactive Elements**: Smooth scrolling, parallax effects, hover states

### Technical Features
- Responsive design for all devices
- CSS animations and micro-interactions
- Smooth scrolling navigation
- Form validation
- Optimized performance
- Accessibility considerations

## File Structure
```
dentist-landing-page/
├── index.html          # Main HTML structure
├── styles.css          # Complete stylesheet with animations
├── script.js           # Interactive functionality
└── README.md          # Documentation
```

## Usage
Simply open `index.html` in a web browser to view the landing page. All styles and scripts are included inline for easy deployment.

## Customization
- Update contact information in the HTML
- Modify colors using CSS variables in `:root`
- Replace placeholder images with actual dental practice photos
- Adjust animations by modifying CSS keyframes
- Customize content for specific dental practice

## Browser Support
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design works on mobile and desktop
- Graceful degradation for older browsers
