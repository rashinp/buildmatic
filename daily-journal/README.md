## Usage

1. **Set Your Mood**: Click one of the mood buttons to reflect your current state
2. **Log Activities**: Add activities using the input field and + button
3. **Write Entry**: Use the journal textarea for detailed thoughts and reflections
4. **Save**: Click "Save Today's Entry" to store your journal entry
5. **Review**: Browse previous entries in the sidebar and click to view details
